# The Flushing A

Official website repository for [theflushinga.com](https://theflushinga.com)

### Structure
```
/index.html     → main site homepage  
/CNAME          → custom domain file  
/assets/        → images, menus, media  
/scripts/       → style.css and script.js  
/img/           → optional image directory
```

### Deployment
1. Upload this folder to GitHub repository `theflushinga-dotcom`  
2. Enable GitHub Pages in repo settings (`main` branch → root)  
3. Ensure the domain DNS at Squarespace points to GitHub's IPs  
4. Once DNS propagates, your site will load directly at https://theflushinga.com
